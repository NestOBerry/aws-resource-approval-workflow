# Contributing

## Project Status

This project is **archived** and not actively maintained. It was developed as a proof-of-concept for AWS resource approval workflows.

## Using This Code

You're welcome to:
- ✅ Fork this repository
- ✅ Modify and adapt for your own use
- ✅ Learn from the implementation
- ✅ Use as a starting point for your projects

## No Active Development

- ❌ Pull requests may not be reviewed
- ❌ Issues may not be addressed
- ❌ No new features will be added

## Questions?

If you have questions about the implementation:
1. Review the documentation in the repository
2. Open an issue (responses may be delayed)
3. Fork and experiment on your own
